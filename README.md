# wechat-mini-program-movies-app
- 拥有以下页面：
  - 首页：随机展示当前热门电影的某一条影评。
  - 电影列表页：展示当前最热门的若干部电影（至少5部）。
  - 电影详情页：展示电影详情信息。
  - 影评列表页：展示某电影的影评列表。
  - 影评详情页：展示某条具体的影评内容。
  - 影评编辑页：用户能在此页面编辑自己的影评内容。
  - 影评预览页：预览已编辑完成的影评。
  - 个人中心页：展示用户已收藏和已发布的影评列表。
